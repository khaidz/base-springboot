# base-springboot
- Framework: SpringBoot 2.7.16
- Xem các thông tin cấu hình trong file application.properties

Các chức năng cơ bản:
- Đăng ký
- Đăng nhập
- Quên mật khẩu
- Kích hoạt tài khoản
- Thay đổi mật khẩu
- Danh sách người dùng
